Django ZODB

Using Django and ZODB together

Dependencies

  • Django >= 1.1
  • ZODB3 >= 3.9.3
  • RelStorage >= 1.2.0 need tests with 1.4.0b1
  • MySQL Python >= 1.2.3c1

Install

$ # Download and install Django.
$ easy_install ZODB3
$ easy_install RelStorage
$ easy_install MySQL-python
$ wget http://github.com/triveos/django-zodb/tarball/master/ -O django-zodb.tar.gz
$ tar xf django-zodb.tar.gz
$ python setup.py install
